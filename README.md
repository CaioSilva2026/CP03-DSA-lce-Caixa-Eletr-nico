## Objetivo do Projeto:

Desenvolver um sistema funcional de caixa eletrônico utilizando a linguagem C, permitindo ao usuário realizar operações bancárias básicas através de uma interface textual organizada.

## Descrição do sistema:
O sistema simula o funcionamento de um caixa eletrônico bancário, permitindo interação do usuário via terminal.

## Funcionalidades implementadas:
- Menu interativo no terminal
- Consulta de saldo
- Saque de valores
- Depósito bancário
- Validação de operações
- Controle de saldo
- Encerramento seguro do sistema

## Como Compilar e Executar

### 1. Clone o repositório

git clone https://github.com/CaioSilva2026/CP03-DSA-lce-Caixa-Eletr-nico.git


### 2. Acesse a pasta do projeto

cd CP03-DSA-lce-Caixa-Eletr-nico

### 3. Compile o código

Caso utilize GCC:

gcc main.c -o caixa


### 4. Execute o programa

No Windows:

caixa.exe

No Linux/macOS:

./caixa
